Response Example

üí¨ Messaging

Send various types of messages and media

POST /chats/{chat_jid}/messages
Send text message

Request Body

{ "text": "Hello, World!", "reply_to_message_id": "message_123" }

Response Example

{ "success": true, "message": "Message sent successfully", "data": { "message_id": "BAE1234567890ABCDEF", "timestamp": "2025-01-01T12:00:00Z" } }
POST /chats/{chat_jid}/images
Send image message

Content-Type

multipart/form-data

Form Data

Parameter Type Required Description
file file Yes Image file (JPEG, PNG, etc.)
caption string No Image caption
view_once boolean No View once media
reply_to_message_id string No Reply to message ID

Response Example

{ "success": true, "message": "Image sent successfully", "data": { "message_id": "BAE1234567890ABCDEF", "media_url": "https://example.com/media/image.jpg", "timestamp": "2025-01-01T12:00:00Z" } }
POST /chats/{chat_jid}/polls
Create and send poll

Request Body

{ "question": "What's your favorite color?", "options": ["Red", "Blue", "Green", "Yellow"], "max_answers": 1, "reply_to_message_id": "message_123" }

Response Example

{ "success": true, "message": "Poll created successfully", "data": { "message_id": "BAE1234567890ABCDEF", "poll_id": "poll_123", "timestamp": "2025-01-01T12:00:00Z" } }
